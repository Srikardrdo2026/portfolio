Projects

Expense Tracker Web Application

This is a full-stack web application built using Python and Flask. It helps users track their daily expenses and monitor monthly budgets in real-time. The application features a secure login and registration system powered by JWT (JSON Web Tokens) for token-based authentication. MongoDB Atlas is used as the cloud database for storing transaction data efficiently. Users can categorize expenses, view summaries, and maintain financial discipline. The backend uses RESTful APIs for smooth communication between frontend and server. Data is stored in a structured NoSQL format, ideal for flexible and scalable design. Built with clarity and simplicity, it can be extended with charts and user dashboards in the future.

JARVIS – Smart Desktop Assistant

JARVIS is a modular voice-based desktop assistant built using Python, inspired by Iron Man's AI assistant. It can recognize voice commands using the SpeechRecognition library and respond via pyttsx3 text-to-speech. JARVIS can open apps, search the web, read the time, access AI responses using the OpenAI API, and more. Designed for offline use and smart local command execution, it brings automation to the desktop. It uses a modular architecture, making it easy to add or remove functionalities (like weather, jokes, etc). The assistant combines user-friendly interactions with backend logic to perform real-world tasks. Future updates can include GUI, dynamic command training, and cloud integration. Ideal for learning how Python interacts with hardware, APIs, and speech technologies.
